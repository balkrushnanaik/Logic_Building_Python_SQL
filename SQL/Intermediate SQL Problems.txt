Here are some **intermediate-level SQL problem statements** focused only on **Basics of SQL, Queries, and Operations (SELECT, WHERE, ORDER BY, GROUP BY, HAVING, JOIN, Aggregate Functions, Subqueries, INSERT, UPDATE, DELETE)**.

### 1. Employee Management

Create queries to:

1. Display all employees whose salary is greater than ₹50,000.
2. Find employees working in the "IT" department.
3. Display the top 5 highest-paid employees.
4. Count the number of employees in each department.
5. Find departments having more than 10 employees.

---

### 2. Student Database

Create queries to:
6. Display students scoring more than 80 marks.
7. Find the average marks of all students.
8. Display the highest and lowest marks.
9. Count students from each city.
10. Find students whose names start with 'A'.

---

### 3. Product Inventory

Create queries to:
11. Display products with stock less than 20.
12. Find the total value of inventory (Price × Quantity).
13. Display products sorted by price in descending order.
14. Find the most expensive product.
15. Increase the price of all products by 10%.

---

### 4. Orders and Customers

Create queries to:
16. Display all orders placed in the current month.
17. Find the total order amount for each customer.
18. Display customers who have placed more than 5 orders.
19. Find customers who have never placed an order.
20. Display the latest order of each customer.

---

### 5. SQL JOIN Operations

Using Employee and Department tables:
21. Display employee names along with department names.
22. Find employees without any department assigned.
23. Display all departments even if no employee works in them.
24. Count employees in each department using JOIN.
25. Find the department with the highest number of employees.

### 6. Aggregate Functions

26. Find the average salary department-wise.
27. Display departments where average salary exceeds ₹60,000.
28. Find the total salary paid by each department.
29. Display the second highest salary.
30. Find employees earning above the average salary.

### 10. Mixed Intermediate Problems
