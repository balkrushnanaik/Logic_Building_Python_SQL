# SQL Basic Practice Problem Statements (50+)

These problems will help you clear SQL fundamentals like:

* `CREATE TABLE`
* `INSERT`
* `SELECT`
* `WHERE`
* `ORDER BY`
* `GROUP BY`
* `HAVING`
* `JOINS`
* `NULL`
* `AGGREGATE FUNCTIONS`
* `SUBQUERIES`
* `UPDATE`
* `DELETE`

* Find students from Pune

* Sort students by marks in descending order
